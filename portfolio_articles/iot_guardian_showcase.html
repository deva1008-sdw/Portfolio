1. Executive Summary & Goal

In security training platforms, users need direct visual feedback on active exploits. However, exposing live devices to full attack payloads carries excessive real-world risks.

IoT Guardian Pro solves this by simulating network attack vectors safely in sandbox loops (representing DDoS flooding, brute-forcing, and phishing intercept nodes) and rendering live network diagnostics tables onto React dashboards via WebSockets.

2. System Architecture

The security administrator initializes attacks from a React GUI, triggering python socket loops in the FastAPI backend, which stream metrics back to live graphs and trigger document exports.

# FastAPI router intercepting simulated phishing entries
@app.post("/api/phish/intercept")
async def intercept_phishing_payload(payload: PhishPayload, db: Session = Depends(get_db)):
    # Record parameters securely in db
    new_entry = PhishIntercept(
        target_username=payload.username,
        source_ip=payload.ip,
        timestamp=datetime.utcnow()
    )
    db.add(new_entry)
    db.commit()
    
    # Broadcast to admin WebSocket dashboard
    await ws_manager.broadcast_message({
        "event": "phish_captured",
        "data": {"username": payload.username, "ip": payload.ip}
    })
    return {"status": "captured"}
